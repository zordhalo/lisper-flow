## Documentation Index

This folder provides comprehensive documentation for the Lisper-Flow repository.

### Core Guides
- `ARCHITECTURE.md` - system architecture and data flow
- `CONFIGURATION.md` - settings reference and configuration examples
- `MODULES.md` - module-by-module overview
- `STREAMING.md` - real-time streaming pipeline details
- `TEXT_INJECTION.md` - typing/injection strategies
- `AUDIO_CAPTURE.md` - audio capture, VAD, and buffering
- `ASR_LLM.md` - ASR and LLM providers
- `HOTKEYS_UI.md` - hotkeys, tray UI, and settings window
- `REFERENCE.md` - file map and repo reference
- `DEVELOPMENT.md` - build, run, debug notes
- `TROUBLESHOOTING.md` - common issues and fixes

### How to Use This Doc Set
Start with `ARCHITECTURE.md` for the high-level flow, then drill down into
`MODULES.md` and the domain-specific docs (`STREAMING.md`, `TEXT_INJECTION.md`, etc.).
